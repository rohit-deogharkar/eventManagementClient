# EventHub Frontend

React + Vite + Tailwind CSS frontend for the EventHub event management app.

For full project setup, API documentation, and deployment instructions, see the [root README](../README.md).

## Quick Start

```bash
npm install
cp .env.example .env
npm run dev
```

App runs at **http://localhost:5173**

Ensure the backend is running at `http://localhost:5000` before using the app.
